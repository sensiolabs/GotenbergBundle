GotenbergBundle
===============

.. caution::

    To use this bundle, you first need to install and configure `Gotenberg 7.x`_.

Overview
--------

The `GotenbergBundle`_ package help you to interact with a Gotenberg API
for Symfony-based projects. Features includes:

* Generate a PDF file from URL
* Generate a PDF file from twig template
* Generate a PDF file from Markdown
* Convert an Office document into PDF
